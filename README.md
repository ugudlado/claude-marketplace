# ugudlado Claude Marketplace

A collection of Claude Code plugins by Mahesh.

## Getting Started

Add this marketplace to Claude Code:

```bash
/plugin marketplace add ugudlado/claude-marketplace
```

Then install any plugin:

```bash
/plugin install lens
```

## Plugins

| Plugin | Description |
|--------|-------------|
| [lens](https://github.com/ugudlado/lens) | A web dashboard to scan, browse, and edit all Claude Code configuration surfaces across all scope levels |
